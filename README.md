# Password Generator

A secure and customizable password generator made in Python.

## Feautures
- Generate multiple password at once
- Choose password length
- Include or exclude:
  - Digits (0-9)
  - Lowercase letters (a-z)
  - Uppercase letters (A-Z)
  - Special symbols (!#$%&*+-=?@^_)
- Option to exclude ambiguous characters like `l`, `1`, `O`, `0`

## How to Run
1. Save your code as 'password_generator.py'
2. Open terminal or Colab
3. Run:

   `python password_generator.py`
   
4. Follow the prompts to generate passwords!
